# Skaitomanas Frontend

„Skaitomanas“ projekto naudotojo sąsaja, sukurta naudojant React ir Vite.

## Technologijos

- **React** + **Vite** (TypeScript)
- **Mantine UI** (Komponentų biblioteka)
- **React Router** (Navigacija)
- **Tabler Icons** (Ikonos)

## Paleidimas

1. Įdiegti bibliotekas:
   ```bash
   npm install
   ```
2. Paleisti vystymo serverį:
   ```bash
   npm run dev
   ```
3. Sukurti produkcinę versiją:
   ```bash
   npm run build
   ```

## Funkcionalumas

- **Responsive dizainas**: Prisitaiko prie mobiliųjų įrenginių (Hamburger meniu).
- **Autentifikacija**: Prisijungimo ir registracijos formos.
- **Knygų katalogas**: Tinklelio išdėstymas su animacijomis.
- **Modalinis langas**: Detali knygos informacija.
- **Tamsi tema**: Palaikoma pagal nutylėjimą (Mantine).
